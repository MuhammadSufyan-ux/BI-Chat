# 💬 BI CHAT - Real-Time Chat Application

A modern, full-stack real-time chat application built using the **MERN Stack** (MongoDB, Express.js, React, Node.js) and **Socket.io**. This platform enables instantaneous communication with a fluid user interface, secure authentication, and robust data management.

---

## ✨ Features

* **Real-Time Messaging:** Instant message delivery and synchronization powered by Socket.io.
* **Full-Stack Architecture:** Robust backend handling API routes and a responsive React frontend.
* **User Authentication:** Secure login and signup workflows for protected user sessions.
* **Live Connection Status:** Real-time tracking of active/online users within the app.
* **Clean & Responsive UI:** Fully optimized design that looks great on both mobile and desktop screens.

---

## 🛠️ Tech Stack

* **Frontend:** React.js, HTML5, CSS3, JavaScript (ES6+)
* **Backend:** Node.js, Express.js
* **Database:** MongoDB
* **Real-Time Engine:** Socket.io (WebSockets)
